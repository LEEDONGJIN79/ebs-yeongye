# 수능특강 독서 — 연계 변형 문제

EBS 2027 수능특강 국어영역 독서 **72개 지문**을 연계·변형한 학습용 웹앱입니다.

## 구성
- **1부 교과서 개념학습** — 1강 독서의 본질 · 2강 독서의 방법 · 3강 독서의 분야 · 4강 독서의 태도
- **2부 적용학습** — 인문·예술 · 사회·문화 · 과학·기술 · 주제 통합
- **3부 실전학습** — 1회 · 2회

각 지문은 **연계 개념 설명 → 3세트 변형 문제 → 유사 수능·모평 풀이** 순서로 이어집니다.

## 사용 / 배포
정적 사이트입니다. `index.html`이 진입점이며, GitHub Pages 등 어떤 정적 호스팅에도 그대로 올려 쓸 수 있습니다.
로컬에서 바로 볼 때는 `앱 실행.bat`(Windows · Python 필요)을 실행하면 됩니다.

## ⚠️ 주의
변형 지문과 문제는 모두 **AI가 생성**한 것이라 오류가 있을 수 있습니다. 감각을 익히는 **가벼운 용도**로만 사용하시길 권합니다.
